Description

Note: Since an OscatsItemBank is also an OscatsAdministrand, it can be tagged with characteristics and can also serve as an item group for multi-stage testing or testlets, with appropriately coded algorithms.

OscatsItemBank does not implement oscats_administrand_get_default_model(), oscats_administrand_set_model(), or oscats_administrand_get_model(). For oscats_administrand_set_default_model(), OscatsItemBank will set the default model for all items.
