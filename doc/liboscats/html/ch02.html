Dependencies

OSCATS depends the following packages:

  • GNU Scientific Library (required)
  • Glib 2, including GObject (required)
  • pkg-config, when using native C (optional)
  • PyGObject module, for Python bindings
  • Perl Glib module, for Perl bindings
  • PHP-GTK extension, for PHP bindings
